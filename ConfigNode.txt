
Instalación de dependencias para el server:

npm init -y
npm i --save-dev babel-cli babel-preset-env
npm i --save-dev babel-watch
npm i express
npm i mongoose
npm i cors

Dentro del package.json agregar dentro de "scripts", debajo de "test" la siguiente línea de código:

"dev": "babel-watch server.js"

Para correr el servidor usen:

npm run dev
